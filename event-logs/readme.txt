These are some example files supporting the process mining book "W.M.P. van der Aalst. Process Mining: Data Science in Action. Springer-Verlag, Berlin, 2016. http://www.springer.com/978-3-662-49850-7"

Also note that many additional event logs (including real-life event logs) are available via:

- https://www.tf-pm.org/ see https://www.tf-pm.org/resources/logs
- https://data.4tu.nl/ see https://data.4tu.nl/repository/collection:event_logs
- http://www.processmining.org/ see http://www.processmining.org/event_logs_and_models_used_in_book
- https://www.coursera.org/learn/process-mining
- www.vdaalst.com

Note that the references to figures etc. refer to the first version of the book. Therefore, the numbers do not match (increment chapter number by 1). The subfolder names refer to the correct chapters, but inside the folders things refer to the chapters of the first version.
